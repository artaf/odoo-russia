Этот модуль предназначен для добавления характеристик к товарам.

Можно создать группы характеристик. В нашем случае группами являются, например:
- тумбы
- душевые кабины
- унитазы

т.е. по сути группа характеристик привязывается к типу товара.

К каждой группе характеристик добавляются характеристики, например такие как:
- цвет
- форма
- мощность

и так далее. Потом у каждого товара выбирается принадлежность к одной из групп характеристик, после чего выбираются
характеристики, для которых имеются значения по этому товару, и эти значения вносятся в базу.

Значения уникальные в пределах одной группы характеристик, т.е. если уже заполнили характеристику Цвет значением Синий,
то в будущем можно уже не вводить вручную, а выбрать Синий из списка.
